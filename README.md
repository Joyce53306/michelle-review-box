# 📦 Michelle's Review Box

Michelle 的 AI 國中學習複習盒 — 五科 × 三年級

## 科目
📖 國文 · 🔤 英文 · 🌏 社會（歷史／地理／公民）· 🔢 數學 · 🔬 自然

## 功能
- 📝 **出題測驗** — 8種題型：綜合、選擇、填空、配對、情境、圖表、排序、簡答
- 💡 **觀念解釋** — 4種風格：故事情境、步驟條列、比較對照、圖表輔助
- ✅ **批改作答** — 貼上作答，AI 給分並解說錯誤原因
- 📊 **學習紀錄** — 自動記錄每次練習（存在瀏覽器）

## 檔案
```
michelle-review-box/
├── index.html   主頁面
├── style.css    樣式
├── app.js       程式邏輯
├── data.js      五科三年級課程資料庫
└── README.md    說明
```

## 使用
直接用瀏覽器開啟 `index.html`，或部署到 GitHub Pages。
